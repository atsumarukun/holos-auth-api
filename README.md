## ルール


### コミットメッセージ

以下の表に記載するprefixを記載する.

| prefix | content |
| --- | --- |
| create | 機能作成 |
| update | 機能更新 |
| remove | 機能削除 |
| refactor | リファクタリング |
| fix | 不具合修正 |

```
create: 新規機能を作成.
```
